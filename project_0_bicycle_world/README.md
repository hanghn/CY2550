# Project 0 — Bicycle World (Linux Introduction)

Beginner Linux exercise navigating a bike-shop themed directory tree. Practiced essential command-line operations: file/directory creation, navigation, permissions, redirection, and basic shell scripting via shebangs.

**Skills practiced:** Bash navigation, file/directory manipulation, `chmod` permissions, output redirection (`>`, `>>`), simple shell scripts
**Tools:** `cd`, `ls`, `mkdir`, `touch`, `chmod`, `echo`, `cp`, `mv`, `cat`
**Files:** `Project_0_Bicycle_World_Linux_Introduction.pdf` (assignment), `bicycle-world/` (work tree), `bicycle-world.zip` (original archive), `hello.txt`
