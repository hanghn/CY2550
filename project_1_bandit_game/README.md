# Project 1 — Bandit Game (OverTheWire Linux CTF)

Solved progressive levels of [OverTheWire's Bandit](https://overthewire.org/wargames/bandit/), a multi-level Linux CTF that teaches command-line skills through password-protected challenges. Each level requires recovering a password to SSH into the next.

**Skills practiced:** SSH, file inspection, encoding/decoding (Base64, hex), `grep`/`find`/`awk`, network tools (`nc`, `ssh`), file permissions, archive handling
**Files:** `flags.txt` (collected level passwords), `token.txt`
