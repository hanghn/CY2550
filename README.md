# CY 2550 — Foundations of Cybersecurity

Coursework from **CY 2550 (Foundations of Cybersecurity)** at Northeastern University, Spring 2026. Eight progressive projects covering Linux fundamentals, CTF challenges, password cracking, cryptography, access control, threat hunting, and social engineering.

| # | Project | Topic | Skills |

| 8 | [CMD CTF](./project_8_cmd_ctf/) | Command-line CTF | Text search, file forensics, encoding |

## Skills demonstrated across the repo

- **Linux & command line** — Bash, file permissions, redirection, shell scripting, system tools
- **CTF problem-solving** — multi-step challenges combining encoding, file analysis, and shell-tool chaining
- **Offensive security tooling** — Hashcat, John the Ripper, OverTheWire wargames
- **Defensive analysis** — KQL threat hunting, security telemetry interpretation
- **Cryptography fundamentals** — classical ciphers, encoding schemes, cryptanalysis basics
- **OSINT & social engineering** — open-source intelligence gathering, pretext development
- **Documentation & process** — every project has a README with topic, skills, and artifacts
