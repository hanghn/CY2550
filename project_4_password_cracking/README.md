# Project 4 — Password Cracking

Cracking hashed passwords from a Unix `shadow`-style file using wordlists and rule-based mutations. Demonstrated how weak password practices and inadequate hashing impact crackability.

**Skills practiced:** Hashcat / John the Ripper workflows, wordlist + rule attacks, hash identification
**Tools:** Hashcat, John the Ripper, `rockyou.txt` + custom wordlists, hashcat rules (`best64.rule`)
**Files:** `cracked.txt` (recovered passwords), `email.txt` (target context)
