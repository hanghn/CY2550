# Project 5 — Discretionary Access Control

Multi-level Linux challenge exploiting Discretionary Access Control (DAC) misconfigurations — improper file permissions, ownership, setuid binaries — to read protected files and escalate privileges.

**Skills practiced:** Unix permissions (`chmod`, `chown`), setuid / setgid behavior, ACL inspection, privilege-escalation patterns
**Files:** `flags.txt` (level passwords), `email.txt` (target context)
